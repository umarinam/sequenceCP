Play on two devices

One player creates a room and shares its code. The second player opens this same file, enters the code, and joins. The host controls the official game state and validates every move.

Create a room

Start PeerJS and wait for one opponent to connect.

Join a room

Paste the room code shown on the host’s device.

PeerJS uses its cloud server only to introduce the two browsers; gameplay data is then sent over WebRTC. Both devices need internet access to load PeerJS and the card images. For best results, serve this file over local HTTP rather than opening it directly as file://.

Quick rules

  • Make two lines of five chips.
  • J♣ / J♦: place on any open card.
  • J♥ / J♠: remove an unlocked enemy chip.
  • Four corner spaces are free for both players.
  • Chips in a completed sequence are locked.
